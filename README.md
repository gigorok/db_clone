# DbClone

This gem allows you to clone your development database.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'db_clone'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install db_clone

## Usage

``` 
rake db:clone[your_new_database_name]
```
